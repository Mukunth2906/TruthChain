# TruthChain
### Blockchain-Based AI Content Provenance & Accountability System

TruthChain is a blockchain simulation that records digital content origin, AI-likelihood, and verification actions in a tamper-evident chain. Every piece of content submitted gets a permanent, verifiable record â€” who wrote it, whether it was AI-generated, and whether it has been tampered with.

---

## ğŸš© Problem

Online content is fundamentally untrustworthy. We cannot answer basic questions like:

- **Who actually created this content?**
- **Was it written by a human or generated by AI?**
- **Has it been silently modified after publication?**

Misinformation, AI-generated spam, and tampered articles erode trust across media, education, and healthcare industries.

---

## âœ… Solution

TruthChain addresses this by storing content metadata on a cryptographically chained ledger. Each block contains:

| Field | Description |
|---|---|
| Content Hash | SHA-256 fingerprint of the content |
| Author Identity | Who submitted the content |
| AI Probability Score | Confidence from HuggingFace model |
| Content Type | AI-generated / Human-written |
| Verification Status | Whether a fact-checker approved it |
| Wallet Signature | ECDSA-signed by the creator |
| Previous Hash | Links to the prior block for tamper detection |

> Any post-publication modification to a block will **break the chain integrity**, making tampering immediately detectable.

### 1. Clone the repository

```bash
git clone https://github.com/Mukunth2906/TruthChain.git

```

### 2. Install dependencies

```bash

```

### 3. Set up your API key

Copy the example env file and add your HuggingFace API token:

```bash

```

Edit `.env`:


### 4. Run TruthChain

```bash
python truthchain.py
```
